# FinTrack API

FinTrack API is a backend-first personal finance REST API built with ASP.NET Core. The goal of this project is to demonstrate backend engineering fundamentals including API design, data modeling, testing, and cloud deployment.

## Project Purpose

This project was built as a portfolio piece to demonstrate skills relevant to backend and fintech software engineering roles. It focuses on building a clean, scalable API architecture rather than UI development.

## Tech Stack

- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / LocalDB
- Swagger / OpenAPI
- xUnit Testing
- Azure App Service
- Azure SQL Database

## Features (MVP)

Transaction management system:

- Create transactions
- View transactions
- Update transactions
- Delete transactions
- Transaction summaries
- Category tracking
- Income vs Expense classification

## Entity Model

### Transaction

Properties:

- Id
- Description
- Amount
- Date
- Category
- Type (Income / Expense)
